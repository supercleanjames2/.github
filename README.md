# VRFL Workflow Templates

GitHub workflow templates for the Virtual Robot Football League (VRFL) platform.

## Available Templates

### VRFL Deploy Workflow
Automated build and deployment workflow for VRFL applications.
- Node.js 20.x setup
- Dependency installation
- Build process
- Production deployment

### VRFL CI Workflow
Continuous integration workflow with:
- Linting
- TypeScript type checking
- Build verification
- Artifact upload

## Usage

These templates will appear in the "Actions" tab when creating a new workflow in any repository within your organization.
